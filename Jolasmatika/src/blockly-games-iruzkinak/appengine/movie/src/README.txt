blocks.js: jokoan erabiltzen diren blokeak eta hauekin erlazionatutako JavaScript kodea sortzeko funtzioak biltzen ditu.

html.js: jokoaren orriaren egitura duen fitxategia.

main.js: jokoan erabiltzen diren aldagaiak (erabiltzaileak sortutako animazioaren "frame" kopurua, adibidez) biltzen ditu eta, beste hiru fitxategiak baliatuz, jokoa hasieratzen du.

scrubber.js: pantailan marrazkiak irudikatzeko funtzioak biltzen dituen fitxategia. Honi esker, erabiltzaileak sartutako aginduek sortutako irudia irudikatu daiteke.
