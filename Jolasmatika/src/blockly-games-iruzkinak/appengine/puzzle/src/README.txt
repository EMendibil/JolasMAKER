blocks.js: jokoan erabiltzen diren blokeak eta hauekin erlazionatutako JavaScript kodea sortzeko funtzioak biltzen ditu.

data.js: jokoan erabiltzen diren puzzle piezen informazioa biltzen du. Pieza bakoitzaren informazio guztia JSON objektu batean gordetzen da, eta hauekin lista bat osatzen da.

html.js: jokoaren orriaren egitura duen fitxategia.

main.js: jokoan erabiltzen diren aldagaiak biltzen ditu eta, beste hiru fitxategiak baliatuz, jokoa hasieratzen du.
