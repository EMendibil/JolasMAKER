avatar.js: ahate bakoitzarekin lotutako aldagai (ahatearen itxura eta posizioa, besteak beste) eta funtzioak  dituen fitxategia

battle.js: "pond" jokoaren "battle" bertsioan erabiltzen diren aldagai eta funtzio ugari biltzen ditu. Orokorrean, 
           borrokan zehar gertatzen diren gauza gehienak (tiroak kudeatu eta ahateen posizioak eguneratu, adibidez)
           eragiten dituzten funtzioak biltzen ditu.

blocks.js: ADI! Blokeen inguruko funtzioak dituen fitxategi zaharkitua. Orain erabiltzen den fitxategia "js-blocks.js" da.

html.js: jokoaren orriaren egitura duen fitxategia.

js-blocks.js: jokoan erabiltzen diren blokeak eta hauekin erlazionatutako JavaScript kodea sortzeko funtzioak biltzen ditu.

pond.js: jokoan erabiltzen den aingira sortzeko beharrezkoak diren aldagai eta funtzioak biltzen dituen fitxategia da.

visualization.js: ahateak eta aingira irudikatzeko behar diren funtzioak dituen fitxategia. Jokoan erabiltzen diren soinuak ere hemen 
                  kudeatzen dira.
